# Parser: yaml/yaml

Invariant: `parser:ryeos/core/yaml/yaml` parses a whole YAML file into a mapping for descriptor-like item kinds.

Most bundle descriptors use this parser, including configs, services, node records, protocols, handlers, parsers, runtimes, and YAML tools.
