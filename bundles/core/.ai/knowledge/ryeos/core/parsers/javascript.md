# Parser: javascript

Invariant: `parser:ryeos/core/javascript/javascript` extracts Rye metadata constants from JavaScript and TypeScript source files.

It is bound to `handler:ryeos/core/regex-kv` and supports `.js`, `.ts`, `.mjs`, and `.cjs` tool formats. It is intended for tool metadata, not general JavaScript parsing.
