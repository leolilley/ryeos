# Tool: ryeos/core/fetch

Invariant: the fetch tool is the local CLI-compatible resolver/reader for items and delegates implementation to the unified `ryeos-core-tools` binary.

It is the tool counterpart to the fetch service and is useful in local/offline contexts where the daemon service path is not used.
