# Handler: yaml-document

Invariant: `yaml-document` parses an entire file body as a YAML mapping.

It is the generic parser for YAML descriptor items: kind schemas, handlers, parsers, protocols, services, configs, node records, runtimes, and YAML tools. Consumer-specific validation happens after parsing.
